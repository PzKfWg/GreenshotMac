# Ralph Loop: Annotation Style Bar Consistency & Functionality

## Mission

Tu es un expert Swift/AppKit qui audite et corrige le système d'annotations de GreenshotMac. À chaque itération, tu dois:

1. **Identifier** les incohérences, bugs, ou options manquantes dans la style bar pour chaque outil d'annotation
2. **Corriger** les problèmes trouvés (code + tests)
3. **Vérifier** que le build compile et les tests passent (`swift build && swift test`)
4. **Committer** chaque correction atomiquement

## Contexte du Projet

GreenshotMac est un éditeur d'annotations pour captures d'écran macOS. L'architecture:

- **AnnotationProtocol.swift** — enum `AnnotationTool` avec des flags `supports*` qui contrôlent la visibilité des options
- **EditorWindowController.swift** — `updateStyleControls(for:style:)` montre/cache les contrôles selon les flags
- **ToolSidebarView.swift** — barre latérale gauche avec les outils
- **Style Bar** — barre horizontale en haut avec les options contextuelles par outil
- **Annotations/** — implémentations individuelles de chaque annotation
- **Filters/** — PixelateFilter, HighlightFilter, ObfuscateFilter

2. **Toujours compiler** avant de committer: `swift build`
